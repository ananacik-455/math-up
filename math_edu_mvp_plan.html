🎮 Core gameplay

Adaptive difficulty engine Must have

Questions get harder as the student succeeds. Tracks accuracy per topic and adjusts difficulty in real time. The heart of the product — without this, it's just a quiz.

ELO-style rating per topic Wrong answer → easier follow-up Streak detection
Math mini-games Must have

3–4 game formats for MVP: speed round (solve before timer), drag-and-match, fill-in-the-blank, and multiple choice. Keeps sessions fresh and engagement high.

Speed round Drag & match Fill in the blank Multiple choice
XP, levels & streaks Must have

Students earn XP per correct answer and level up. Daily streaks encourage return visits. Visual rewards (stars, badges) give immediate feedback — critical for ages 10–18.

XP per answer Level-up animation Daily streak counter Achievement badges

📚 Content & curriculum

Topic library — grades 4–11 Must have

Structured topic tree aligned to the Ukrainian school curriculum. MVP should cover arithmetic, fractions, algebra basics, geometry, and equations — the most NMT-tested areas.

Arithmetic Fractions & percentages Algebra Geometry basics Equations
NMT practice mode Must have

Dedicated mode with real NMT-style questions (timed, multiple choice, no hints). A huge differentiator for Ukrainian students preparing for the national exam.

NMT question format Timed sessions Topic coverage map
Step-by-step hints Should have Paid

When a student is stuck, they can unlock a hint showing the solving method step by step. Free tier gets 3 hints/day; paid gets unlimited. Strong upsell trigger.

📊 Progress & profile

Progress dashboard Must have

Shows mastery per topic (e.g. "Fractions: 74%"), total XP, current level, and weak spots. Students and parents can see exactly what needs work.

Per-topic mastery % Weak spot highlight Weekly activity graph
Leaderboard Should have Free

Weekly class or grade-level leaderboard inside Telegram. Drives social competition and organic sharing — Telegram's social graph makes this very powerful.

Weekly top-10 Share score to chat

⚙️ Telegram mini app tech stack

Telegram-specific requirements

The app lives inside Telegram as a Web App (TWA). Uses Telegram's native auth — no separate login needed. Payments via Telegram Stars or direct card.

Telegram Web App SDK Telegram auth Telegram Stars Reminders bot Share to chat
Frontend

React + Vite. Telegram UI Kit for native feel. Optimised for mobile — all interactions must be thumb-friendly.

Backend

Node.js or Python (FastAPI). PostgreSQL for user progress. Redis for leaderboard. Hosted on Railway or Render for fast start.

🗓 Suggested build order

Week 1–2 — skeleton

Telegram bot + Mini App shell, auth, 1 game mode (multiple choice), 20 questions from 2 topics, XP counter.

Week 3–4 — core loop

Adaptive difficulty, daily streaks, progress dashboard, full topic library (arithmetic + algebra MVP content).

Week 5–6 — differentiation

NMT mode, leaderboard, 3 additional game formats, hints system with paywall, Telegram Stars payment integration.

Week 7–8 — polish & launch

Animations, onboarding flow, reminder bot messages, performance optimisation, beta test with 20–30 real students, fix critical bugs.
